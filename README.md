# StyleShuffle: Версия с Встроенным Backend во Flutter

## 📜 Описание

**StyleShuffle** — интеллектуальное кроссплатформенное мобильное приложение для управления гардеробом и подбора одежды. Пользователи могут загружать фотографии одежды,
автоматически классифицировать их, создавать стильные образы и учитывать погодные условия. Интеграция с онлайн-магазинами позволяет примерять и приобретать одежду.

В этой версии приложения отсутствует выделенный сервер. Весь backend реализован с использованием **Firebase** и встроен в клиентскую часть на **Flutter/Dart**. 

## 🛠️ Технологии

- **Flutter/Dart**: фронтенд и встроенный backend.
- **Firebase Authentication**: управление пользователями и безопасная авторизация.
- **Firebase Firestore**: облачное хранилище для данных гардероба, нарядов и пользовательских предпочтений.
- **Firebase Storage**: хранение изображений одежды.
- **API**: интеграция с внешними сервисами (например, погодные API и удаление заднего фона).
- **Python**: обучение сверточной нейронной сети для обработки изображений.

## 📁 Структура проекта

### Assets

В папке `assets` хранятся модели сверточных нейронных сетей и необходимые текстовые данные:

- `model_unquant.tflite` и `model_unquant2.tflite`: обученные модели для анализа изображений.
- `labels.txt` и `labels2.txt`: данные для классификации изображений.

### Основные экраны

- **TakePhotoPage.dart**: экран для съёмки или загрузки фотографии одежды.
- **CollectionSetPage.dart**: экран для просмотра сохранённых образов.
- **CreateSetPage.dart**: создание новых образов.
- **WeatherPage.dart**: экран текущей погоды.
- **SingInPage.dart** и **RegistrationPage.dart**: экраны для регистрации и входа.
